# ailiveX 平台頁面設計規格書

給設計師的全站頁面清單與功能需求。  
設計師基於此文件輸出前後台完整 UI/UX，供工程師實作。

---

## 一、整體背景

**ailiveX** 是一個 AI 角色互動平台，讓用戶與被指派的 AI 角色進行文字或即時語音對話，角色會記住用戶（記憶系統），並能代為生成策略書、企劃書等正式文件。

**兩種使用者身份：**
- **用戶（User）**：使用前台，只看到自己被指派的角色，與角色互動
- **管理員（Admin）**：使用後台，管理角色、用戶、權限、記憶

---

## 二、前台頁面（用戶端）

### P-01 登入頁 `/login`

**目的：** 帳號密碼登入，進入對應身份首頁

**功能：**
- 帳號輸入欄位
- 密碼輸入欄位
- 登入按鈕（loading 狀態）
- 錯誤提示（帳號或密碼錯誤）
- 登入後自動跳轉（管理員 → 後台首頁；用戶 → 大廳）

**設計要點：**
- 品牌識別感強，有角色氛圍
- 不需要「忘記密碼」（帳號由管理員統一建立）

---

### P-02 大廳 `/lobby`

**目的：** 用戶看到自己被指派的全部角色，選一個進入對話

**功能：**
- 角色卡片 Grid：頭像（方圓形）、角色名稱、「可語音通話」標籤（有語音 ID 才顯示）
- 點擊角色卡 → 進入文字聊天頁
- 頂端導航：「我的文件」連結、登出按鈕
- 空狀態：「還沒有被指派任何角色，請聯絡管理者」

**設計要點：**
- 卡片有懸停效果，點擊感明確
- 角色頭像是整個頁面的視覺重心
- 最終會有多個角色；RWD，手機版單列

---

### P-03 文字聊天 `/chat/[characterId]`

**目的：** 與角色進行文字對話，核心互動介面

**功能區塊：**
- **頂端欄**：角色頭像 + 名稱；語音通話按鈕（有語音才顯示）；返回大廳連結
- **對話區（主體）**：
  - 對話泡泡列表，用戶靠右（自己），角色靠左
  - 角色訊息附小頭像
  - 自動捲動到最新訊息
  - 支援換行（Markdown 渲染，但不顯示符號）
- **輸入區（底部固定）**：
  - 文字輸入框（多行，Enter 送出，Shift+Enter 換行）
  - 送出按鈕
  - 傳送中 loading 狀態（角色思考中...）
- **文件卡片**：角色回覆中若包含生成文件，顯示「文件已建立」提示卡片，附連結到文件頁

**設計要點：**
- 氛圍感強，不是冷冰冰的客服介面
- 角色頭像在對話中持續存在感
- 輸入區要固定在底部，對話區可獨立捲動

---

### P-04 即時語音通話 `/realtime/[characterId]`

**目的：** 真人般即時語音對話（LiveKit）

**功能區塊：**
- **全螢幕視覺**：角色大頭像/形象圖置中；音波視覺化效果（角色說話時有動態）
- **連線狀態指示器**：
  - 六個健康度燈號：Token、麥克風、房間、Agent、音訊、網路
  - 每個燈號有三態：準備中（灰）/ 正常（綠）/ 異常（紅）+ 說明文字
- **通話控制**：
  - 靜音/取消靜音按鈕
  - 掛斷按鈕（掛斷後返回聊天頁或大廳）
- **診斷日誌（摺疊/展開）**：開發用，可隱藏於生產環境
- **通話中狀態**：連線中 / 通話中 / 角色說話中（動態提示）

**設計要點：**
- 角色形象是全頁主視覺，背景可有氛圍光影
- 音波動效是核心體驗，建議整合到角色頭像周圍（光暈/脈衝）
- 介面要極簡，通話時不分心
- 進入此頁即自動開始連線，無需額外按鈕

---

### P-05 我的文件 `/documents`

**目的：** 查看角色代為生成的策略書、企劃書等文件

**功能：**
- 文件列表（依建立時間排序，最新在上）
- 每筆：文件標題、建立時間、生成狀態、操作按鈕
- **狀態標籤**：
  - 排隊中（等待生成）
  - 撰寫中（LLM 生成中）
  - 排版中（轉 HTML）
  - 完成（可查看）
  - 失敗（紅色，說明）
- **操作按鈕**：
  - 完成狀態 → 「查看」（在新分頁開啟 HTML）
  - 完成狀態 → 「下載 PPT」（下載 .pptx）
  - 未完成 → 狀態顯示，無操作按鈕
- 頁面每 5 秒自動刷新進行中的文件狀態
- 空狀態：「還沒有文件，在對話中告訴角色幫你寫一份」
- 頂端：返回大廳連結

**設計要點：**
- 表格/列表，乾淨易讀
- 狀態有進度感（動態 loading 點）
- 生成中的文件讓用戶感覺「正在發生」

## 三、後台頁面（管理員端）

### A-01 後台首頁 `/admin`

**目的：** 管理員進入後的導航中樞

**功能：**
- 四張導航卡片：
  1. **角色** — 建立角色（靈魂 + 頭像）
  2. **用戶** — 開帳號發給用戶
  3. **指派** — 決定誰能跟哪些角色聊
  4. **記憶** — 查看 / 調層 / 刪除記憶
- 頂端：平台名稱 + 登出按鈕

**設計要點：**
- 功能清單明確，進入每個子功能快速
- 可考慮每張卡片加數字摘要（幾個角色、幾個用戶）

---

### A-02 角色管理 `/admin/characters`

**目的：** 建立與編輯 AI 角色

**功能：**
- **左側：角色列表**
  - 每筆：角色名稱、頭像縮圖、是否有語音（標籤）
  - 點擊 → 右側展開編輯表單
  - 「新增角色」按鈕 → 展開空白表單
- **右側：編輯表單**
  - 基本資訊：角色名稱
  - **靈魂（soulCore）**：多行文字框，大面積，這是最重要的設定
    - 說明文字：「用角色能理解的語言描述它的個性、說話方式、世界觀」
  - **頭像上傳**：拖拉或點擊上傳，預覽縮圖，支援 GCS 上傳
  - **語音設定**（可選，展開/摺疊）：
    - Minimax 語音 ID（文字輸入）
    - 速度滑塊（0.5 - 2.0）
    - 音高滑塊（-12 - 12）
    - 音量滑塊（0.1 - 2.0）
    - 情感選項（neutral / happy / sad / ...）
  - 儲存按鈕 / 刪除按鈕

**設計要點：**
- 靈魂欄要夠大，讓管理員可以寫很長的角色設定
- 語音設定收摺，主視覺焦點是靈魂文字

---

### A-03 用戶管理 `/admin/users`

**目的：** 建立用戶帳號，供用戶登入前台

**功能：**
- **新增區**（上方）：
  - 帳號（username）
  - 顯示名稱
  - 密碼（≥ 6 碼，明文輸入，後端 scrypt 加密）
  - 送出按鈕
  - 成功/錯誤提示
- **用戶列表**（下方）：
  - 每筆：顯示名稱、帳號、建立時間、角色 role（user/admin）
  - （暫不支援修改密碼或刪除，可標示「未來功能」）

**設計要點：**
- 簡潔表單，管理員快速開帳號
- 列表有排序（最新在上）

---

### A-04 權限指派 `/admin/access`

**目的：** 控制每個用戶能看到哪些角色

**功能：**
- 上方：選擇用戶（下拉選單）
- 下方：角色勾選列表
  - 每筆：角色頭像 + 名稱 + 勾選框
  - 勾選即新增權限，取消勾選即移除
  - 即時儲存（無需按鈕）或單一「儲存」按鈕
- 「全選」快捷鈕

**設計要點：**
- 選完用戶後角色列表立即更新當前狀態
- 操作回饋要清楚（哪個勾了、哪個沒勾）

---

### A-05 記憶管理 `/admin/memories`

**目的：** 讓管理員查看、調整、刪除各用戶×角色的記憶

**功能：**
- **篩選列**：
  - 用戶選擇（下拉）
  - 角色選擇（下拉）
  - 記憶分層（all / fresh / core / archive）
  - 記憶類型（all / fact / emotion / preference / promise / question / milestone）
- **記憶列表**（表格）：
  - 欄位：類型標籤、內容（截斷 + 展開）、分層、重要性（1-10）、點擊次數、建立時間
  - 每筆：可切換分層（下拉）、刪除按鈕
- **批量操作**：
  - 全選 / 取消全選
  - 批量晉升 Tier（fresh → core）
  - 批量刪除

**設計要點：**
- 記憶類型有顏色標籤（6 種：fact/emotion/preference/promise/question/milestone）
- 內容欄可展開，避免截斷重要資訊
- 批量操作區域清楚分開

## 五、設計補充需求

### 品牌感

- 平台名稱：**ailiveX**
- 調性：沉浸感、AI 存在感、不是工具感
- 角色是有靈魂的，不是聊天機器人
- 用戶與角色之間有「關係」（記憶幾次對話、認識多久）

### 設計系統基礎（現有 CSS 變數）

```css
--bg          背景色
--panel       卡片/面板底色
--panel-2     次層面板
--border      邊框
--accent      主強調色
--accent-2    次強調色
--muted       輔助文字
--radius      圓角（卡片）
```

### RWD 優先度

| 頁面 | 優先 |
|------|------|
| 登入頁 | 手機優先 |
| 大廳 | 手機優先 |
| 文字聊天 | 手機優先 |
| 即時語音 | 手機優先（全螢幕體驗） |
| 我的文件 | 桌面為主（表格） |
| 後台所有頁面 | 桌面為主 |

### 介面語言

- 前台：繁體中文（或依角色語言設定）
- 後台：繁體中文

```
登入頁
  ├── 管理員 → 後台首頁 → 角色管理 / 用戶管理 / 指派 / 記憶管理
  └── 用戶   → 大廳
                ├── 文字聊天 ──→ 即時語音通話
                │                   └── 掛斷 → 返回聊天
                └── 我的文件
```
